## 📱 Loyiha haqida

**OpportunityHub UZ** — O'zbekiston yoshlari uchun hackathonlar, bepul kurslar, grantlar, tanlovlar va stajyorlik imkoniyatlarini bir joyda to'playdigan mobil-friendly veb ilova.

### ✨ Imkoniyatlar

| Tur | Tavsif |
|-----|--------|
| 🏆 Hackathon | Milliy va xalqaro hackathonlar |
| 📚 Bepul Kurslar | Sertifikatli onlayn ta'lim |
| 🌟 Grantlar | Stipendiyalar va moliyaviy yordam |
| 🥇 Tanlovlar | Dasturlash va ijodiy musobaqalar |
| 💼 Stajyorlik | To'lov qilinadigan amaliyot |

## ⚙️ Firebase sozlash

1. [Firebase Console](https://console.firebase.google.com) ga kiring
2. Yangi loyiha yarating yoki mavjudini tanlang
3. Authentication → Providers → Google va GitHub ni yoqing
4. `firebaseConfig` ni `index.html` ga qo'shing
5. Authorized domains ga saytingiz domenini qo'shing
